# Bauernhof-App

Interaktive HTML-Lern-App für den Grundschul-Unterricht zum Thema Bauernhof-Tiere, Namen, Artikel und Quiz.

- `index.html` enthält die komplette App (HTML, CSS, Inline-SVGs, JavaScript).
- Keine externen Abhängigkeiten außer Google Fonts.
- Keine Secrets enthalten.

Live:
- https://einhorn.live/bauernhof/
- https://einhorn.live/bauernhof-test/
